## 🛠️ Toolkit

### Reconnaissance
- **[theHarvester](https://github.com/laramies/theHarvester)** — email, subdomain & host enumeration from public sources
- **[SpiderFoot](https://github.com/smicallef/spiderfoot)** — automated OSINT reconnaissance across multiple data sources
- **[Sherlock](https://github.com/sherlock-project/sherlock)** — username hunting across hundreds of platforms

### Username & Email Investigation
- **[Blackbird](https://github.com/p1ngul1n0/blackbird)** — fast username & email investigation across social platforms
- **[Holehe](https://github.com/megadose/holehe)** — check where an email has been used to register accounts
- **[GHunt](https://github.com/mxrch/GHunt)** — locate Google Maps accounts and extract data from Drive links
- **[Epieos](https://epieos.com)** — check if an email has an associated Google Maps presence
- **[Whatsmyname](https://whatsmyname.app)** — web-based username search across major platforms

### Search Techniques
- **Google Dorking** — advanced search operators to surface hidden or exposed data
- **Pastebin Analysis** — scanning paste sites for leaked credentials and sensitive data
- **[OSINT Framework](https://osintframework.com)** — curated directory of categorised OSINT resources

### Geolocation
- **Google Earth / Google Maps** — satellite and street-level imagery analysis
- **[Overpass Turbo](https://overpass-turbo.eu)** — geographic queries against OpenStreetMap
- **[Mapillary](https://www.mapillary.com)** — crowdsourced street-level imagery
- **Reverse Image Search** — Lenso, PimEyes for image-based geolocation
